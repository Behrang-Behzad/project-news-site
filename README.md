# News Site

Build a news web site using HTML and CSS. Try using grid or flex box and media queries. Animation if possible.

## The problem

I divided the page in 3 sections, Header, main and footer. The main part has the main content for 12 stories on cards and a side bar for ad space. The cards are animated on hover.

The header section has the Navigation list on top and a scrolling animation for breaking news. The scrolling section disappears on smaller screens.

Main space is a grid that is arranged differently based on how wide the screen is.
